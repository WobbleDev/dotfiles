Search URL: http:///?q=%s. Please do not include prefix http:// and suffix /?q=%s. For example, if you input www.someurl.com and your search keyword is bingo, the search request URL generated will be http://www.someurl.com/?q=bingo, which will be accepted by most search engines. When you click save for the first time, Chrome will ask you about permission. Because we cannot dynamically add a permission for a single website, we have to include all websites. However, we did our best to use as few permissions as possible, which is http://*/?q=*.
